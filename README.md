# DWDM Optical Networks – Lambda Visualization

## Descripción
Proyecto demostrativo orientado a la visualización de lambdas (longitudes de onda)
utilizadas en redes ópticas DWDM mediante una interfaz web.

El objetivo del proyecto es mostrar de forma gráfica y didáctica cómo se organizan
y utilizan las longitudes de onda en un entorno de transmisión óptica.

## Características
- Visualización de lambdas DWDM
- Representación gráfica de canales ópticos
- Enfoque educativo y demostrativo
- Interfaz web simple

## Tecnologías
- HTML
- CSS
- JavaScript

## Uso
Proyecto orientado a fines demostrativos y de aprendizaje sobre redes ópticas DWDM.
